## Sample Exercises Repository 
This private repository contains **representative sample exercises** 

The notebooks provided here are **illustrative examples only**, intended to give an overview of the **scope, depth, and computational flavor** of AI tools in computational chemistry.

---
## Repository Scope

This repository contains a **subset of instructional notebooks** demonstrating:

- Core machine learning workflows  
- Molecular and materials representations  
- Supervised and unsupervised learning methods  
- Model interpretability and analysis  

These exercises are **not exhaustive** and do not represent the final course deliverables.

---
## Tools and Execution

The notebooks are designed to run using:

- Python  
- scikit-learn  
- PyTorch  
- RDKit  
- DScribe  
- Jupyter Notebook or Google Colab  

No proprietary software is required.
